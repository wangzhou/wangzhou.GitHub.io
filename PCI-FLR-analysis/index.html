PCI FLR analysis

发表于 2021-06-29 更新于 2022-07-24 阅读次数： Valine：
PCI协议里有function level reset的定义，实现FLR的设备可以提供function级别的复位。 本文梳理linux内核里和FLR相关的流程。本文可以作为设备复位设计的一个参考。

关于FLR的硬件操作比较简单, 相关的硬件有:

  • 配置空间里device cap里的FLR capability bit, 这个表示设备是否支持FLR。
  • 配置空间里device control里的BCR_FLR bit, 写这个bit可以触发FLR。
